How to Use & Deploy

Deploying on Google Apps Script

  1. Open script.google.com and create a new project.
  2. Go to File → New → HTML file and create three files: index, styles, and script.
  3. Paste the contents of index.html into index.html. For styles.html, wrap the CSS in <style>…</style>. For script.html, wrap the JS in <script>…</script>.
  4. In index.html, replace the <link> tag with <?!= include('styles') ?> and the <script src="script.js"> tag with <?!= include('script') ?>.
  5. In Code.gs, add a doGet function:
    function doGet() { return HtmlService.createTemplateFromFile('index').evaluate().setTitle('Whiskered').setSandboxMode(HtmlService.SandboxMode.IFRAME); }
  6. Add a helper: function include(f) { return HtmlService.createHtmlOutputFromFile(f).getContent(); }
  7. Go to Deploy → New deployment, choose Web App, configure access, then click Deploy.

Customizing Tab Order

In script.js, find the SITES array near the top. Each entry has an id, label, icon, and tagline. Reorder or rename entries freely — the HTML tab list, panels, and side panel all stay in sync automatically.

Customizing Colors

In styles.css, the :root block at the top contains --clr-light-orange and --clr-orange. Changing these two values recolors the entire theme. Fine-tune --clr-orange-dark, --clr-orange-mid, and --clr-orange-pale for more control.

Customizing Animations

The :root block also exposes --dur-fast (180ms), --dur-med (280ms), and --dur-slow (350ms) for transition durations, plus --ease-out and --ease-spring for easing curves. Increase values for slower, more deliberate motion; decrease for snappier feedback.
